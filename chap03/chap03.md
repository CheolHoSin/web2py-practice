# 제어문

제어문은 프로그램 실행 흐름을 바꿈

조건에 따라 분기를 나누거나, 동일한 실행을 반복한다

## Boolean

`True` or `False`

## Operator

- Expression
  - +, -, *, /, ...
- Comparison Operator: It returns boolean
  - ==, <, >, ...
- Membership Operator: It returns boolean
  ```python
  print('world' in 'Hello world') # True
  ```

## 조건문

```
if *True or False*:
    *Statement*
```
